# Adaptive Front Lighting System

## Overview
This project implements an Adaptive Front Lighting System (AFLS) for automobiles using embedded electronics. The system automatically adjusts vehicle headlights based on steering direction and environmental conditions to improve visibility and safety during night driving.

---

## Features
- Automatic headlight direction control
- Steering angle-based lighting adjustment
- Improved night driving visibility
- Embedded system-based implementation
- Low power consumption

---

## Components Used
- Microcontroller
- Servo Motor
- Sensors
- LEDs / Headlights
- Power Supply Circuit

---

## Software Used
- Embedded C / Arduino IDE
- Proteus / TINA Simulation
- KiCad / PCB Design Software

---

## Applications
- Smart automotive lighting
- Advanced driver assistance systems
- Vehicle safety systems
