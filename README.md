# Adaptive Front Lighting System

## Overview

The Adaptive Front Lighting System (AFLS) is an embedded automotive safety project designed to improve nighttime driving visibility. The system dynamically adjusts the direction of vehicle headlights based on steering movement to provide better illumination during turns.

---

## Features

- Automatic headlight direction control
- Steering angle-based adjustment
- Improved corner visibility
- Embedded system implementation
- Low power operation
- Automotive safety enhancement

---

## Working Principle

The system detects steering movement and rotates the headlights using a servo motor mechanism. As the vehicle turns, the headlights automatically adjust their angle to illuminate the road ahead more effectively.

---
